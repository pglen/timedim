Introduction:

   Welcome to the TimeDim Copy Machine. This application was born out of the need for an easy to use backup and/or backup automation. TimeDim can be used to copy any source directory to any target directory. The TimeDim Copy Machine only copies files that are newer than the target.

 History:
The original copy machine was a DOS and UNIX command line utility, and we have used it for a decade, with great success. In some instances, we have used TimeDim as often as every hour.


   Theory of operation:

TimeDim operates on directories. It evaluates every file and subdirectory from that directory, and all subdirectories thereof, and will update the target directory accordingly. TimeDim reads the modification date of the source file and the target file, and it will copy the source file to the target if the source is newer than the target. TimeDim will assume that a non existing file is older than the source, so it will copy files that are missing from the target.
    Legacy.

    TimeDim is the GUI version of the infamous dcp32.exe For completeness, we included the old command line version, which was named 'dcp32.exe' at the time. (Directory Copy for 32 bit Windows)
