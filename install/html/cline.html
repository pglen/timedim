TimeDim Command Line syntax

Command line usage:
c:> timedim SourceDir TargetDir [[option1] ... [option]]
c:> timedim /f  sessionfilename

Option Descriptions:

Option
 Option Name
 Description
/v
 verify Verify file after copying.
/b
 backup Backup old target file if it was to be overwritten.
/s
 simulate Simulate copy. Do not copy, but show what would be copied
/p
 prompt Prompt the user before the copy starts.
/f filename
 session_file Execute a previousely saved session.




TimeDim can be used from the command line in two modes:
  • 1.) dir mode
  • 2.) session mode.
In dir mode one needs to specify both source and darget directories. In session mode, one may specify a previosely saved session file.
