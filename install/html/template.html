Introduction:

    Welcome to the TimeDim Copy Machine. This application was born out of the need for an easy to use backup and/or backup automation. TimeDim can be used to copy any source directory to any target directory. The Copy Machine only copies files that are newer than the target.

   History:

   The original copy machine was a DOS and UNIX command line utility, and I have used it for a decade, with great success. In some instances, I have used it as often as every hour.
